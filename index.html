i Formatting tips For reliable verse and Q/A prompt splitting, use these markers consistently: v1, v2… (or bare 1, 2… or 3:16) to start each verse in Verse mode. Q1, Q2… to start each question, and Answer (or A1, A2…) right after it in Q & A mode. Multi-part numbering like 1) 2) 3) carries through to both the question and the answer, for example: Q1 1) Who said it? 2) To whom?
Answer
1) Charlie 2) The unicorns Other formats like "1.", "(1)", or "[1]" aren't recognized — sticking to the markers above keeps the split predictable.

Library sync file

Keep the library in a single file you own. Point it at a folder your own sync service already watches — iCloud Drive, Dropbox, OneDrive, Syncthing — and your devices stay in step, while Verbatim still talks to no server. Verbatim saves to the file automatically after every change, and merges from it on startup. The file doubles as a readable backup; entries you encrypted stay encrypted inside it.

Install Verbatim on this device

Installing adds Verbatim to your device like a normal app — its own icon, its own window, fully offline — and unlocks the private on-device library. Nothing goes through an app store, and nothing changes about privacy: it's this same page, kept on your device.

Your browser supports one-click install.
Chrome / Edge / Brave (desktop)
Look at the right end of the address bar for an install icon (a screen with a down-arrow in Chrome, ⊕ in Edge) and click it — or open the browser menu ⋮ and choose “Cast, save and share → Install page as app…” (Chrome) / “Apps → Install this site as an app” (Edge). Verbatim then opens from your dock, taskbar, or launcher in its own window, with the Library available.

Safari (Mac)
With Verbatim open, choose File → “Add to Dock…”. It appears in your Dock as an app.

iPhone / iPad (Safari)
Tap the Share button, then “Add to Home Screen”. Open Verbatim from the new icon.

Android (Chrome)
Open the menu ⋮ and tap “Install app” (or “Add to Home screen”).

Firefox (desktop)
Firefox doesn't install web apps on desktop. Use one of the browsers above for the installed app — the website still works here for practice and sharing.

After installing, open Verbatim from its own icon — that's where the Library lives.
Activates saving on this device without a full install (for Firefox, or when install is being stubborn).
